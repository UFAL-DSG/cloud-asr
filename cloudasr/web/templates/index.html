CloudASR software platform and a public ASR webservice

CloudASR is a software platform and a public ASR webservice. Platformâ€™s API supports both batch and incremental speech recognition. The batch version is compatible with Google Speech API.

Features

Scalability

CloudASR is designed to be able to run on multiple machines and it can be scaled dynamicaly according to the users need. Finally, according to the conducted benchmarks CloudASR is able to handle 1000 parallel requests.

Customizability

Because CloudASR is built on top of Kaldi speech recognition toolkit it can run almost any acoustic and language models created by Kaldi. Furthermore, CloudASR can be customized to support any other ASR engine.

Easy Deployment

CloudASR can be run on a single machine with Docker or it can be run with Mesos on AWS or any other cloud provider. CloudASR uses one configuration file and can be started with single command.

Contact Us

If you have any questions regarding CloudASR you can reach us at our mailinglist: cloudasr@googlegroups.com.
